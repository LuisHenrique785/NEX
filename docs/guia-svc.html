🏭 Papel: Log SVC

Diferença entre Agência e SVC

A Agência é um ponto de coleta: faz inventário local e expede os pacotes para o SVC com dados do motorista.

O SVC é o hub central: recebe os pacotes expedidos pelas agências, confirma os volumes e atualiza o status de cada pacote para "recebido no SVC".

📦
Seção 02
Recebendo Pacotes das Agências

Quando um caminhão chega com pacotes de uma agência, registre todos os pacotes. Use o método mais prático para o momento.

📷
Câmera / Scanner
Aponte a câmera para o QR Code ou código de barras. Registro imediato sem confirmação extra.
⌨️
Digitação Manual
Digite o código do pacote e confirme. Ideal quando o código não escaneia.
🖼️
Foto do Código
Fotografe o código de barras ou QR Code e o sistema lê automaticamente.
⚡

Dica de velocidade

Para lotes grandes, o modo câmera é o mais rápido. Posicione bem o celular e escaneie em sequência — o sistema adiciona automaticamente sem precisar confirmar cada código. Duplicatas são alertadas de forma automática.

📋
Seção 03
Passo a Passo: Receber um Lote
1

Confirme a chegada do veículo

Antes de escanear, confirme com o motorista: qual agência enviou os pacotes e quantos volumes estão no manifesto.

2

Toque em "Receber Pacotes"

No painel SVC, toque em Receber Pacotes. A tela de recebimento abrirá.

3

Preencha os dados do motorista (opcional)

Nome e placa são opcionais mas ajudam no rastreamento. Preencha se possível para facilitar análise de divergências no painel Consulta.

4

Escaneie todos os pacotes

Use câmera, digitação ou foto. Cada leitura bem-sucedida adiciona o código à lista. Você pode escanear centenas de pacotes em sequência sem parar.

5

Atenção a duplicatas

Se um pacote já foi registrado, o sistema alertará automaticamente. Não registre o mesmo código duas vezes — pode indicar erro no manifesto.

6

Toque em "Confirmar Recebimento"

Um modal de confirmação aparecerá com o total de pacotes. Confirme para gravar. O app volta automaticamente ao painel SVC após a confirmação.

✅

Após confirmar

Todos os pacotes do lote são gravados no banco. O app navega de volta ao painel SVC automaticamente. As agências que expediram esses pacotes verão o status atualizado para "Recebido no SVC" na tela de inventário delas.

🚨

Não libere o veículo com divergência

Se o número de pacotes escaneados for diferente do manifesto, não libere o veículo. Confira fisicamente os volumes e registre qualquer divergência antes de liberar.

Seção 04
Inventário de Sacas

O SVC também registra a movimentação de sacas — as bolsas usadas para transportar pacotes. Existem dois tipos de registro:

Operação Quando usar Dados necessários
✅ Chegada Quando sacas chegam ao SVC das agências Quantidade + Transportadora + Placa
🚚 Expedição Quando sacas saem do SVC Quantidade + Placa + Transportadora
✅

Rastreabilidade completa

Todos os registros ficam salvos com data e hora em BRT (Brasília). O histórico de sacas do dia é exibido na tela com as movimentações mais recentes no topo.

🔍
Seção 05
Painel de Consulta

O painel Consulta oferece análise completa da operação: expedições, status por agência, análise de motoristas, busca de pacotes e exportação de dados. Acessível diretamente do menu SVC.

1

Toque em "Consulta" no painel SVC

No painel principal do SVC, toque no botão 🔍 Consulta. Também pode ser acessado diretamente da tela inicial.

2

Insira a senha de acesso

O painel pede senha de acesso. Consulte o responsável pelo sistema para obter a senha.

3

Navegue pelas 5 abas

O painel tem 5 abas de análise. Cada aba mostra um aspecto diferente da operação.

🔍 Consulta — Painel de Análise
📦 Expedições
🏪 Por Agência
🚛 Motoristas
🔍 Buscar
📥 Exportar

📦 Expedições: Lista as últimas 60 expedições com resumo — quantos pacotes foram enviados, quantos chegaram no SVC e quantos estão pendentes.

🏪 Por Agência: Agrupa pacotes por agência, período de 7, 15 ou 30 dias. Mostra IDs e permite download CSV por agência.

🚛 Motoristas: Análise de divergência por motorista — quantos pacotes pegou vs entregou no SVC.

🔍 Buscar Pacote: Rastreie qualquer código — inventário, expedição, recebimento SVC.

📥 Exportar CSV: Exporta inventário completo por período com todas as colunas.

📊

Para análise detalhada

O guia do Time MELI contém documentação completa do painel Consulta com todos os detalhes de cada aba, filtros disponíveis e como interpretar os dados.

Seção 07
Fluxo Diário do SVC

Rotina recomendada

Início do turno: Verifique o painel Consulta — aba Expedições — para ver se há pacotes expedidos pelas agências que ainda estão aguardando recebimento no SVC.

Durante o turno: A cada chegada de veículo, toque em "Receber Pacotes", preencha os dados do motorista (opcional mas recomendado) e escaneie todos os pacotes antes de liberar o veículo.

Confirmar: Sempre confirme via o botão "Confirmar Recebimento" — o app mostrará um modal de confirmação e depois voltará automaticamente ao painel SVC.

Final do turno: Consulte a aba Motoristas para verificar se há divergências — motoristas que pegaram mais pacotes do que entregaram.

  • Verificar painel Consulta ao iniciar o turno
  • Registrar cada chegada antes de liberar o veículo
  • Preencher dados do motorista quando possível
  • Confirmar totais com o manifesto físico
  • Não liberar veículos com divergência de volume
  • Checar aba Motoristas ao final do turno
🏆

Operação zero pendências

O objetivo é que todo pacote expedido pelas agências seja confirmado pelo SVC no mesmo dia. Use o painel Consulta → aba Por Agência para acompanhar a situação de cada NODO.

Seção 08
Problemas Comuns e Soluções
Problema Causa provável Solução
Câmera não abre Permissão negada no browser Configurações do browser → Permitir câmera para o site
Código não escaneia Código danificado ou sujo Limpe o código, tente a foto ou digitação manual
"Já registrado" Pacote já está no lote atual Não registre novamente; verifique se não é duplicata física
Botão confirmar não responde Nenhum pacote adicionado ao lote Adicione ao menos 1 pacote antes de confirmar
App não volta para SVC após confirmar Instabilidade de rede ou cache Atualize a página (F5 / recarregar)
Dados não salvam Sem conexão de internet Verifique o Wi-Fi ou dados móveis; tente novamente
Site não carrega Instabilidade de rede Atualize a página. Se persistir, use dados móveis

Suporte técnico
