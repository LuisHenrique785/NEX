🏭 Papel: Log SVC

O Logístico SVC é o ponto central de recebimento na operação NEX. Você recebe os pacotes que as agências expedem, registra tudo no sistema e garante a rastreabilidade completa da operação.

Seção 01
Entrando no Sistema como SVC
1

Abra o NEX Inventory

Acesse luishenrique785.github.io/NEX no navegador. Você verá a tela inicial com duas opções de perfil.

2

Selecione "SVC"

Na tela inicial, toque no botão SVC. Você entrará diretamente no painel de recebimento de pacotes.

3

Pronto para operar

O painel SVC carrega automaticamente. Não é necessário selecionar NODO — o SVC recebe de todas as agências.

Diferença entre Agência e SVC

A Agência é um ponto de coleta: faz inventário local e expede os pacotes para o SVC.

O SVC é o hub central: recebe os pacotes expedidos pelas agências e garante que chegaram corretamente.

📦
Seção 02
Recebendo Pacotes das Agências

Quando um caminhão chega com pacotes de uma agência, você precisa registrar todos os pacotes no sistema SVC. Existem 3 formas de fazer isso — use a que for mais prática para o momento.

📷
Câmera / Scanner
Aponte para o QR Code ou código de barras do pacote. Registro imediato.
⌨️
Digitação Manual
Digite o código do pacote no campo e confirme. Ideal quando o código não escaneia.
🖼️
Foto do Código
Fotografe o código de barras ou QR Code e o sistema lê automaticamente.
⚡

Dica de velocidade

Para lotes grandes, o modo câmera é o mais rápido. Posicione bem o celular e escaneie um por um em sequência — o sistema adiciona automaticamente sem precisar confirmar cada vez.

📋
Seção 03
Passo a Passo: Receber um Lote
1

Confirme a chegada do veículo

Antes de escanear, confirme com o motorista: qual agência enviou os pacotes e quantos volumes estão no manifesto.

2

Abra o painel SVC no sistema

Toque em SVC na tela inicial. O painel de recebimento será exibido.

3

Selecione o método de entrada

Escolha entre Câmera, Manual ou Foto. A câmera é recomendada para velocidade.

4

Escaneie cada pacote

Passe a câmera sobre o código de barras ou QR de cada pacote. Cada leitura bem-sucedida emite um vibração/som de confirmação.

5

Atenção a duplicatas

Se um pacote já foi registrado, o sistema alertará automaticamente. Não registre o mesmo código duas vezes — pode indicar erro no manifesto.

6

Confirme o total ao final

Compare o número de pacotes registrados com o manifesto físico. Se houver diferença, investigue antes de liberar o veículo.

🚨

Não libere o veículo com divergência

Se o número de pacotes escaneados for diferente do manifesto, não libere o veículo. Confira fisicamente os volumes e registre qualquer divergência antes de assinar o recebimento.

Seção 04
Inventário de Sacas

O SVC também registra o movimentação de sacas — as bolsas usadas para transportar pacotes entre pontos. Existem dois tipos de registro:

Operação Quando usar Dados necessários
✅ Chegada Quando sacas chegam ao SVC Quantidade
🚚 Expedição Quando sacas saem do SVC Quantidade + Placa + Transportadora
1

Acesse "Inventário de Sacas"

No painel SVC, selecione a opção de sacas.

2

Selecione: Chegada ou Expedição

Escolha a operação correta para o momento atual.

3

Para Chegada: informe a quantidade

Digite quantas sacas chegaram e confirme. O registro é salvo automaticamente com data e hora.

4

Para Expedição: preencha todos os campos

Informe: Quantidade, Placa do veículo e Transportadora. Todos são obrigatórios para rastreabilidade.

✅

Rastreabilidade completa

Todos os registros ficam salvos com data, hora e operador. Em qualquer auditoria, é possível consultar o histórico completo de movimentação de sacas.

1

Escaneie o código do pacote

Use a câmera, digitação ou foto para consultar um pacote específico.

2

Veja o histórico do pacote

O sistema mostrará: quando chegou, de qual agência, e se foi expedido.

3

Consulte o painel geral

Seção 06
Gerenciando Pendências

Pacotes do dia anterior que não foram expedidos aparecem automaticamente como pendências no início de cada novo dia.

⏰

Pendências do dia anterior

Ao abrir o sistema pela manhã, verifique os pacotes pendentes. São itens que chegaram mas ainda não foram expedidos. Cada um precisa ser investigado ou expedido.

1

Veja as pendências do dia anterior

A seção "Pendências" lista todos os pacotes recebidos que ainda estão no SVC sem expedição registrada.

2

Investigue cada pendência

Para cada pacote pendente: está fisicamente no SVC? Foi expedido sem registro? Houve extravio?

3

Regularize no sistema

Se o pacote foi expedido: registre a saída normalmente. Se está no SVC: deixe como pendente até expedir.

📊
Seção 07
Fluxo Diário do SVC

Rotina recomendada

Início do turno: Verifique as pendências do dia anterior. Resolva ou registre cada uma antes de iniciar as operações do dia.

Durante o turno: A cada chegada de veículo, registre todos os pacotes antes de liberar. Nunca acumule para registrar depois — o risco de erro aumenta muito.

A cada expedição: Registre a saída com placa e transportadora. Isso fecha o ciclo do pacote no SVC.

Final do turno: Confirme que não há pendências não resolvidas. O que não foi registrado hoje vira pendência amanhã.

  • Verificar pendências do dia anterior antes de iniciar
  • Registrar cada chegada antes de liberar o veículo
  • Conferir totais com o manifesto físico
  • Registrar todas as expedições com placa e transportadora
  • Revisar pendências ao final do turno
  • Acionar o gestor em caso de divergência não resolvida
🏆

Operação zero pendências

O objetivo é terminar cada dia com zero pendências não justificadas. Todo pacote que chega ao SVC deve ter uma saída registrada — ou uma justificativa documentada.

Seção 08
Problemas Comuns e Soluções
Problema Causa provável Solução
Câmera não abre Permissão negada no browser Vá em Configurações do browser → Permitir câmera para o site
Código não escaneia Código danificado ou sujo Limpe o código, tente a foto ou digitação manual
"Já registrado" Pacote já está no sistema Não registre novamente; verifique se não é duplicata física
Dados não salvam Sem conexão de internet Verifique o Wi-Fi ou dados móveis; tente novamente
Divergência de quantidade Erro no manifesto ou extravios NÃO libere o veículo; acione o gestor imediatamente
Site não carrega Instabilidade de rede Atualize a página (F5). Se persistir, use dados móveis
📞

Suporte técnico

Em caso de problemas que impeçam a operação, contate o time MELI responsável pelo NEX. Registre o problema com prints da tela e horário da ocorrência para agilizar o atendimento.
