📷 Scanner Aponte a câmera para o QR Code ou código de barras Uso principal — mais rápido e preciso
⌨️ Digitar Digite o código do pacote manualmente Quando o código estiver danificado ou ilegível

🚫
Pacotes duplicados não são aceitosSe você tentar bipar um pacote que já foi escaneado hoje, o sistema vai avisar e ignorar automaticamente.
⚠️
Atenção: PendênciasPacotes bipados ontem que não aparecerem no inventário de hoje serão exibidos como PENDÊNCIAS no topo da tela. Não se esqueça de inventariá-los!
Passo 4 — Expedição de Pacotes
1
Acesse "Inventário de Pacotes" → "Expedição de Pacotes"
Quando for enviar pacotes com um motorista, use esta opção.
2
Preencha os dados do motorista
Todos os campos são obrigatórios para registrar a expedição.
CampoExemplo
👤 Nome do MotoristaJoão da Silva
🪪 CPF do Motorista123.456.789-00
🚛 Placa do VeículoABC1234
🏢 TransportadoraTotal Express
3
Adicione os pacotes (scanner / digitar / foto)
Use qualquer um dos 3 métodos para adicionar os pacotes que estão sendo expedidos. Após confirmar, os pacotes saem do inventário e não aparecem mais como pendência.
✅
Após a expediçãoOs pacotes expedidos são removidos automaticamente do inventário ativo. Eles não aparecerão mais como pendência para a agência.
